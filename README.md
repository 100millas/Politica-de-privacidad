Política de Privacidad

Última actualización: [fecha]

Esta aplicación/GPT utiliza una integración con Intervals.icu para leer y, en su caso, escribir datos de entrenamiento del usuario con el único fin de analizar la carga de entrenamiento, generar recomendaciones y, si el usuario lo solicita, crear o actualizar entrenamientos planificados en su calendario.

Datos tratados
- Datos de actividades y entrenamiento procedentes de Intervals.icu
- Datos de bienestar o recuperación, si están disponibles
- Datos del perfil deportivo aportados por el usuario
- Datos de planificación introducidos por el usuario en ChatGPT

Finalidad
- Analizar el entrenamiento del deportista
- Generar recomendaciones y planificación
- Crear, actualizar o reemplazar entrenamientos en Intervals.icu cuando el usuario lo solicite expresamente

Base de funcionamiento
La conexión con Intervals.icu se realiza mediante la API correspondiente autorizada por el usuario o por una clave configurada por el titular de esta integración.
